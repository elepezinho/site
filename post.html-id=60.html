Microsoft Revela Segunda Onda de Jogos para o Game Pass em Novembro

A Microsoft surpreendeu os assinantes do Game Pass nesta terça-feira (14) ao revelar a emocionante segunda leva de jogos que chegará ao serviço neste mês de novembro. Entre os destaques está o aguardado Persona 5 Tactica, que estará disponível para os membros no dia do lançamento, oferecendo uma experiência envolvente aos fãs da renomada franquia.

Os jogadores já podem mergulhar em Coral Island, disponível tanto na nuvem quanto no Xbox Series X|S. Em 17 de novembro, Persona 5 Tactica chega para expandir as opções de jogabilidade, disponível não apenas para consoles, mas também para PC, oferecendo uma experiência abrangente aos assinantes do Game Pass.

Além disso, preparem-se para o final do mês, quando Dune: Spice Wars chegará para os usuários de nuvem e consoles, juntamente com Rollerdrome, disponível na nuvem, PC e Xbox Series X|S. A diversidade de títulos garante uma experiência rica e variada para os assinantes do serviço.

Contudo, a notícia vem acompanhada de um aviso triste para alguns jogos que deixarão o catálogo do Game Pass ainda neste mês. Sete títulos, incluindo Anvil, Battlefield 1943, Battlefield: Bad Company, Battlefield: Bad Company 2, Disc Room, Eastward e Grid, darão adeus aos assinantes. É hora de aproveitar ao máximo esses jogos antes que deixem o serviço.

Com essa revelação, a Microsoft continua a oferecer aos assinantes do Game Pass uma ampla variedade de jogos para explorar e desfrutar. Esteja você interessado em aventuras intensas, estratégias emocionantes ou experiências únicas, o Game Pass continua a ser um tesouro para os amantes de jogos, mantendo seu compromisso de fornecer conteúdo de alta qualidade.

Para ficar por dentro de todas as novidades e interagir com a comunidade gamer, a Microsoft incentiva os usuários a explorarem as redes sociais do NerdBunker, incluindo o grupo do Telegram. Mantenha-se conectado para não perder nenhum anúncio ou atualização sobre o mundo dos jogos.
